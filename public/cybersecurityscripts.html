Cyber-security scripts

This project was a dynamic exploration of cybersecurity, focusing on password security, hash functions, and web penetration testing. Key objectives included unraveling the intricacies of brute force attacks, leveraging Python for effective hashing, and evaluating various hash functions. Alongside, I delved into advanced topics such as file manipulation in Python and gained hands-on experience with essential Python tools for cybersecurity.

Within this exploration, I developed Python scripts to enhance my understanding. One script demonstrated the generation of MD5 and SHA-256 hashes, emphasizing the critical role of hash algorithms in securing passwords. Another script explored password entropy, offering a quantitative measure of password strength.

These scripts served as invaluable educational tools, enriching my comprehension of fundamental concepts in password security and reinforcing practical cybersecurity skills. Simultaneously, in web penetration testing, I utilized tools like httpx, dnsx, subfinder, gobuster, ffuf, and gau, contributing significantly to a comprehensive understanding of web penetration testing techniques. The project seamlessly blended theoretical insights with practical applications, fostering a holistic approach to cybersecurity exploration.
